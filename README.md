# Todo-App-Jetpack-Compose
Basic To-Do Application made with Jetpack Compose

### Tech Stack Used
- Jetpack Compose
- Jetpack NavHost Controller
- Hilt Dependency Injection
- Kotlin Flows
- Kotlin Coroutines
- Data Store
- ROOM
